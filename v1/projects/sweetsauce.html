Sweetsauce Apparel

2010 - 2013 / Client / Branding / Apparel

The Script

Sweet Sauce had never really had it’s own logotype before. Sure, it was set and displayed on nearly every shirt, but it was different everytime. While that still works sometimes, we wanted a default logotype to use for those instances in which it was necessary. Ryan, the owner, and I decided on a hand‐drawn script. After a few hours, a lot of doodling, and a few tweaks, the Sweet Sauce script logo was born.

The Mark

The Sweet Sauce robot is something Ryan has been drawing and incorporating into Sweet Sauce since it’s inception (according to him, even before that). Naturally, it was something he wanted to keep around. When he was upgraded to version 2.0, he gained a sharp pixel edge. Still, he doesn’t clash with the more natural script.
